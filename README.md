# Post-Correspondence-Problem-of-sushi

ポスト対応問題を体験できるブラウザゲームです。

## 画面構成

- タイトル画面
  - ゲームモード選択
    - NP-ハードモードへの遷移ボタン
    - 決定不能モードへの遷移ボタン
- NP-ハードモード問題選択画面
  - グリッド上に配置される問題番号
- 問題画面
  - ポスト問題のタイルに対応するボタン（刺身パック＆タンポポのベルトコンベア風UI）
  - 決定不能モードのみ不可能ボタンが追加される
  - 生成された文字列×2の表示エリア
  - 戻るボタン
  - すべて消去するとクリア画面に遷移
- クリア画面
  - 次の問題へ
  - ホームに戻る

## 開発・実行

```bash
npm install
npm run dev
```

## 技術構成

- Vite + React + TypeScript
- Tailwind CSS
